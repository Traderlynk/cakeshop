Cakeshop Plugin Readme

Overview

Cakeshop Plugin enables Openfire to become an Ethereum node.

Known Issues

Installation

  1. Copy the cakeshop.jar file to the OPENFIRE_HOME/plugins directory.
  2. Restart Openfire.
  3. Configure the admin properties page.

Configuration

Under Server settings -> Cakeshop tab you can configure the working folder for the cakeshop.
